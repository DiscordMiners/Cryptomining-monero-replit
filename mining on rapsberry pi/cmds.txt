We will be mining Monero which we will get XMR coins
Req:
	Need a Digital wallet(From getmonero.org) 
	Windows Download Link: https://downloads.getmonero.org/gui/win64install

IN The `GUI Wallet` Go to `Account` and click on the copy icon to copy the wallet address [Required for Step 9] 


Connecting to raspberry pi from windows
1)Open Command Prompt
2)Enter command: ssh pi@[Raspberry pi ip]
3)Enter message: yes
4)Enter password: [Raspberry pi password]

After connecting to raspberry pi
1) sudo apt update
2) sudo apt install git build-essential cmake libuv1-dev libssl-dev libhwloc-dev -y
3) git clone https://github.com/xmrig/xmrig.git 
4) cd xmrig
5) mkdir build
6) cd build
7) cmake ..
8) make
#Make sure u have a wallet by this step
9)./xmrig -o gulf.moneroocean.stream:10128 -u [Wallet address] -p [nameurworker/nickname]
